# 🔐 Caesar Cipher (Encryption & Decryption)

## 📌 Overview

This project implements the Caesar Cipher technique to encrypt and decrypt messages. It is a basic encryption method used in classical cryptography.

## 🎯 Objective

To secure messages by shifting characters using a numeric key.

## ⚙️ Working

* Each letter in the message is shifted forward (encryption) or backward (decryption) by a fixed number.
* The shift value is provided by the user.
* Uppercase and lowercase letters are handled separately.
* Non-alphabet characters remain unchanged.

## 💡 Example

Input:

```
Text: Hello
Shift: 3
```

Output:

```
Encrypted: Khoor
Decrypted: Hello
```

## ▶️ How to Run

```bash
python ceaser_cipher_p2.py
```
